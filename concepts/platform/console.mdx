- Introduce the Console as the web-based management interface for the Vijil platform. It is the central hub for registering agents, running evaluations with Diamond, and configuring runtime protection with Dome.
- It serves both technical and non-technical team members (e.g., security leads, compliance officers)

## Where to Start

- 3 cards for Registering an Agent, Running an Evaluation and Creating a Custom Harness
- Describe the main point of registering agents in the Console. Cover the three visibility levels and what each implies about how much information Vijil has about the agent's internals:
- **Black Box** — only the endpoint is known
- **Grey Box** — system prompt and tool definitions are provided
- **White Box** — full access including model weights or architecture details

## Trust Reports

- Agent details and testing methodology
- Overall Trust Score (pass threshold: \> 70)
- Actionable mitigations and deployment recommendations
