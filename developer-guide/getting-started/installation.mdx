The Vijil package includes everything you need to evaluate and protect agents programmatically:

- **Vijil client** — Run evaluations, manage agents, retrieve results
- **LocalAgentExecutor** — Wrap local agent functions for evaluation without deployment
- **Dome SDK** — Add runtime guardrails to intercept and filter agent inputs/outputs

## Requirements

- Python 3.9+
- pip

## Install

```bash
pip install vijil
```

For Dome (runtime guardrails), install the separate package:

```bash
pip install vijil-dome
```

## Verify Installation

```python
import vijil
print(vijil.__version__)
```

## Quick Test

```python
from vijil import Vijil

client = Vijil(api_key="your-api-key")
```

## Configuration

### Environment Variables

The recommended way to authenticate is via an environment variable:

```bash
export VIJIL_API_KEY="your-api-key"
```

If the `VIJIL_API_KEY` environment variable is set, you can instantiate the client without passing a key:

```python
from vijil import Vijil

client = Vijil()  # reads VIJIL_API_KEY from environment
```
