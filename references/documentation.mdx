## Function: `standalone_function`

A standalone function to add two numbers.

This is a simple function to illustrate how function docstrings outside
of a class are processed.

#### Parameters

- **x** (`x`): The first number.
- **y** (`y`): The second number.


## Function: `greet`

Generates a greeting message.

This method combines the instance's name with a provided greeting
to form a complete sentence.

#### Parameters

- **greeting** (`greeting`): The greeting to use, e.g., "Hello".
